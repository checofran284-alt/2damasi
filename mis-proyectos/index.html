"GEMINI"

La semana del 22 de septiembre se nos fue asignada la tarea de crear un contador de personas, desarrollando un diseño funcional y visual en una sola página o pantalla.

El ejercicio consiste en implementar un sistema que permita contar personas mediante una interfaz gráfica sencilla. En ella se debe mostrar la cantidad actual de personas y distinguir correctamente entre el uso de “Persona” o “Personas”, según el número mostrado.

Además, se deben incluir botones para aumentar (+1) y disminuir (-1) el contador, aplicando restricciones que eviten valores negativos o inconsistentes. También se debe añadir un contador separado para hombres y mujeres, de modo que el programa pueda reflejar ambos valores de forma independiente.

Por último, la interfaz debe tener una imagen de fondo, cuidando la presentación visual para lograr un diseño claro y agradable. Como mejora opcional, se pueden incluir detalles adicionales, como un botón para reiniciar los contadores o mensajes dinámicos según el número de personas registradas.
